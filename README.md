# Usage
```bash
// React
cd frontend
npm install http-proxy-middleware --save  
npm install axios --save  
npm start  
'''

* EC2
'''
// SpringBoot
cd Backend
./gradlew clean build
nohup java -jar build/libs/Backend-0.0.1-SNAPSHOT.jar &

// Flask
cd Backend/src/main/python
pip install -r requests.txt
nohup python3 run.py &
```
